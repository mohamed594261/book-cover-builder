# Book Cover Builder

مشروع لتصميم أغلفة الكتب باستخدام Fabric.js وواجهة تفاعلية


book-cover-builder/
├── src/                # الكود الأساسي
│   ├── js/             # ملفات JavaScript
│   │   ├── modules/    # وحدات منفصلة (canvas-front.js, canvas-back.js, ...)
│   │   └── app.js      # نقطة الدخول الرئيسية
│   ├── css/            # ملفات CSS
│   │   └── style.css
│   ├── assets/         # صور، خطوط، أي موارد ثابتة
│   └── index.html      # صفحة البداية
├── dist/               # ملفات البناء الجاهزة للتوزيع
├── README.md           # وصف المشروع
└── package.json        # إعدادات npm (لو هنستخدم مكتبات)
